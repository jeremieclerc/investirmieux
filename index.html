Les questions les plus fréquentes

Comment fonctionne le simulateur ?

Le simulateur vous permet de trouver le placement le plus adapté à votre profil. Il vous suffit de renseigner le montant que vous souhaitez placer, votre niveau de risque et votre banque. Le simulateur vous proposera alors le placement le plus adapté à votre profil.

Quels sont les frais de ce service ?

Le service est entièrement gratuit. Nous ne prenons aucune commission sur les placements que nous vous recommandons.

Quelles sont les banques disponibles ?

Actuellement, nous travaillons avec Boursorama et Fortuneo. Nous prévoyons d'ajouter d'autres banques dans le futur.
